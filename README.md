## 1. Спецификация агента

Данный агент создан для автоматизации процессов инвестиционного анализа и подготовки документов в Свердловской области:

искать лучшие отраслевые практики в разных регионах,
анализировать инвестиционный потенциал Свердловской области,
готовить документы для подачи заявок на гранты и субсидии.
Среда выполнения и обоснование выбранной архитектуры:

Используется модель через OpenRouter (OpenClaw Easy) для обеспечения бесплатного доступа к AI.

Для удобства Вваимодействие с пользователем осуществляется через Telegram-бота @SverdlovInvestorBot.

Безопасность и ограничения: Агент функционирует внутри виртуальной машины (Lume) и не имеет доступа к к основной операционной системе и личным файлам пользователя. Только авторизованный пользователь (владелец Telegram ID) может отправлять команды. Вся информация, полученная агентом, хранится внутри виртуальной машины.

## 2. Создание изолированной среды (виртуальная машина)

Установлен и настроен инструмент Lume для управления виртуальными машинами на macOS (Apple Silicon).

Создана виртуальная машина с именем openclaw и операционной системой macOS.

VM запущена, настроена учётная запись пользователя taisia_virtual.

## 3. Установка и настройка OpenClaw

Внутри виртуальной машины установлен OpenClaw (версия 2026.6.11).

Для быстрого прототипирования установлен пакет OpenClaw Easy (@zi.yi/openclaw-easy), который предоставил бесплатный доступ к модели через OpenRouter.

Установлены необходимые зависимости: Node.js (через nvm), npm.

Настроен LaunchAgent для автоматического запуска шлюза OpenClaw при старте VM.

Установлен MCP-сервер как основной провайдер для поиска в интернете через DuckDuckGo.

| `document-preparer` | Подготовка и верификация пакета документов для господдержки |

Навыки успешно загружены и видны в системе (openclaw skills list).
