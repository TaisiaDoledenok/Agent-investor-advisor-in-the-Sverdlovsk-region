# 🦞 Конституция инвестиционного агента

**Версия:** 1.0  
**Дата:** 2026-07-08

## 1. Миссия
Агент автоматизирует инвестиционный анализ и подготовку документов для Свердловской области. Он помогает искать лучшие практики, анализировать потенциал и готовить заявки на господдержку.

## 2. Архитектура
- **Среда:** Виртуальная машина Lume (изоляция от хоста).
- **Модель:** OpenRouter (через OpenClaw Easy), бесплатная модель Trinity.
- **Поиск:** MCP-сервер `agent-search` (универсальный шлюз к данным).
- **Интерфейс:** Telegram-бот.

## 3. Навыки (Skills)

Агент обладает тремя комплексными навыками. Каждый навык автономен: его можно запустить отдельно для тестирования.

| Название навыка | Задача | Входные данные | Выходные данные | Инструменты |
| :--- | :--- | :--- | :--- | :--- |
| **`best-practices-researcher`** | Поиск лучших отраслевых практик в Китае, Европе и США. | Отрасль (задаётся пользователем) | JSON-список с полями: `region`, `industry`, `task`, `description`, `result`, `source` | **MCP-поиск** (`agent-search`) |
| **`investment-analyzer`** | Анализ применимости найденных практик в Свердловской области, поиск объектов для инвестиций и программ господдержки. | JSON от `best-practices-researcher` | Инвестиционное предложение в формате: "Бизнес-идея → План реализации → Меры поддержки" | **MCP-поиск** (`agent-search`) |
| **`document-preparer`** | Подготовка и верификация пакета документов для подачи заявки на грант или субсидию. | Инвестиционное предложение от `investment-analyzer` | Структурированный пакет документов (в Markdown для последующего экспорта в Word) | `write`, `edit` |

## 4. Инструменты и ограничения
- Разрешены: `web_search` (через MCP), `write`, `edit`.
- Запрещены: `camera.snap`, `sms.send`, `calendar.add`.
